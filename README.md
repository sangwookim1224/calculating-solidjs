# Calculating

SolidJS로 만든 계산기 앱입니다.

## 기술 스택

- [SolidJS](https://www.solidjs.com/) - UI 프레임워크
- [Vite](https://vitejs.dev/) - 빌드 도구

## 시작하기

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 빌드 결과 미리보기

```bash
npm run preview
```

## 기능

- 사칙연산 (+, -, ×, ÷)
- 소수점 입력
- 부호 전환 (+/-)
- 퍼센트 계산 (%)
- AC / C 초기화
- 0으로 나누기 오류 처리
